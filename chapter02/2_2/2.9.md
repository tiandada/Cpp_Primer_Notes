### (a). 
```
std::cin >> int input_value;
```
expected primary-expression before ‘int’
```
int input_value = 0;
std::cin >> input_value;
```
### (b). 
```
int i = { 3.14 };
```
narrowing conversion of ‘3.1400000000000001e+0’ from ‘double’ to ‘int’ 
```
double i = { 3.14 };
```
### (c).
```
double salary = wage = 9999.99;
```
‘wage’在此作用域中尚未声明
```
double wage;
double salary = wage = 9999.99;
```
### (d).
```
int i = 3.14;
```
会丢失部分值
```
double = 3.14;
```
